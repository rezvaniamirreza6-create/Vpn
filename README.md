## 🚀 دپلوی روی Railway

### ۱. ایجاد پروژه
```
railway new
```

### ۲. افزودن PostgreSQL
در Railway Dashboard → Add Service → PostgreSQL  
متغیر `DATABASE_URL` به صورت خودکار ست می‌شود.

### ۳. ست کردن متغیرها
در Railway → Variables، موارد زیر را اضافه کنید:

```
BOT_TOKEN=توکن_ربات_شما
ADMIN_IDS=آیدی_عددی_شما
PANEL_URL=http://آی‌پی‌پنل:پورت
PANEL_USERNAME=نام_کاربری_پنل
PANEL_PASSWORD=رمز_پنل
INBOUND_IDS=1
BOT_NAME=فروشگاه VPN من
SUPPORT_USERNAME=یوزرنیم_پشتیبانی
ZARINPAL_MERCHANT=کد_مرچنت_زرین‌پال
CARD_NUMBER=شماره_کارت
CARD_HOLDER=نام_صاحب_کارت
REFERRAL_REWARD=50000
```

### ۴. دپلوی
```
railway up
```

## ⚙️ تنظیمات پنل 3X-UI

1. پنل را روشن کنید (نیاز به SSL ندارد)
2. یک inbound بسازید (Vless یا VMess)
3. شماره ID آن inbound را در `INBOUND_IDS` قرار دهید
4. یوزرنیم/پسورد پنل را ست کنید

---

## 🎰 سیستم قرعه‌کشی

- کاربران با زدن دکمه **قرعه‌کشی** یک شماره ۴ رقمی منحصر‌به‌فرد دریافت می‌کنند
- ادمین از پنل خود **قرعه‌کشی ادمین** را می‌زند
- تعداد برنده (۱، ۳ یا ۵ نفر) را انتخاب می‌کند
- ربات به صورت خودکار به همه کاربران نتیجه را اعلام می‌کند

---

## 🎊 سیستم دعوت

- هر کاربر لینک اختصاصی دارد
- وقتی با لینک کسی ثبت‌نام شود، مبلغ پاداش به کیف پول معرف اضافه می‌شود
- ادمین می‌تواند مبلغ پاداش را از **⚙️ تنظیمات ربات** تغییر دهد

---

## 🏷 کدهای تخفیف

- ادمین کد با درصد و محدودیت استفاده می‌سازد
- کاربر هنگام خرید کد وارد می‌کند
- کد به صورت خودکار تعداد استفاده را کم می‌کند و غیرفعال می‌شود
